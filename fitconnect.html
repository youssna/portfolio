FITCONNECT (SportTech)

Écosystème complet de coaching sportif : Administration Java & Plateforme Web.

Présentation du projet

  • Solution hybride permettant de gérer une salle de sport ou un réseau de coachs.
  • Premier projet majeur utilisant Java pour la partie métier.
  • Communication fluide entre une interface lourde (Desktop) et un portail client (Web).

Fonctionnalités hybrides

  • Admin (Java) : Validation des coachs, gestion des clients et suivi des statistiques.
  • Client (Web) : Inscription, choix d'objectif et consultation du programme.
  • Automatisation : Assignation intelligente des programmes selon les objectifs.

Défis techniques

  • Mise en place du pattern DAO (Data Access Object) en Java.
  • Synchronisation d'une base de données unique pour deux environnements différents.
  • Gestion des contraintes d'intégrité SQL lors de la suppression de données liées.

Architecture & Data

  • Utilisation de JDBC pour la connexion Java / MySQL.
  • Interface utilisateur Java Swing avec design personnalisé (FlatLaf).
  • Modélisation relationnelle optimisée pour le matching Coach/Client.
